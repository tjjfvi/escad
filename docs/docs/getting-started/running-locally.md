# Running Locally

## Installation

1. Install [Deno](https://deno.land/#installation)
2. Run

```sh
deno install -Afn escad --unstable --no-check https://escad.dev/cli/start.ts
```

## Usage

1. Run
```sh
escad myFile.ts
```

2. Open [http://localhost:8046/](http://localhost:8046/) in your browser
